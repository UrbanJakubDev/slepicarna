# 🛠️ Slepicarna Tech Stack (Striktní)

## Core
- **Next.js 15 (App Router)**
- **tRPC v11** (API vrstva)
- **Prisma ORM** (Database access)
- **SQLite** (Lokální databáze `dev.db`)

## UI & Analytics
- **Tailwind CSS** + **Shadcn/ui**
- **Lucide Icons**
- **Recharts** (Vizualizace snůšky)

## Auth & Security
- **Custom PIN Login** (Numeric keypad)
- **Cookies** (Persistence 30 dní)

## Infrastructure
- **Bun** (Runtime & Package Manager)
- **Vercel** (Hosting)
